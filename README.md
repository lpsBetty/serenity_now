SerenityNow
===========

Ruby wrapper for [TVRage API][rage].

"Serenity now, insanity later"


Installation
------------

Add this line to your application's Gemfile:

    gem 'serenity_now'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install serenity_now
